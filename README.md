# IP延迟监测工具

一个功能强大的IP延迟和端口状态监控工具，支持实时监控、告警通知、数据可视化和导出功能。

## 功能特性

### 1. 基础监控功能
- 实时监控IP延迟
- 端口状态检测
- MTR路由追踪
- 丢包率统计

### 2. 告警系统
- 延迟告警（可配置阈值）
- 丢包率告警（可配置阈值）
- 端口状态变化告警
- 告警历史记录
- 告警阈值可视化

### 3. 数据可视化
- 延迟趋势图（带告警阈值线）
- 端口状态变化图
- 告警历史趋势图
- 实时数据更新

### 4. 数据导出
- CSV格式导出
- 包含完整监控数据
- 支持按IP分别导出
- 包含时间戳、延迟、丢包率、端口状态等信息

### 5. 用户界面
- 响应式设计
- 标签页式图表展示
- 告警数量徽章
- 实时状态更新
- 简洁直观的操作界面

## 系统要求

- Python 3.7+
- Windows/Linux/MacOS
- MTR工具（用于路由追踪）

## 安装步骤

1. 克隆或下载项目到本地

2. 安装Python依赖：
```bash
pip install -r requirements.txt
```

3. 安装MTR工具：
   - Windows: 下载并安装 [WinMTR](https://sourceforge.net/projects/winmtr/)
   - Linux: `sudo apt-get install mtr` (Ubuntu/Debian)
   - MacOS: `brew install mtr`

## 配置说明

### 告警阈值配置
在 `ip_monitor.py` 中可以调整告警阈值：
```python
self.alert_thresholds = {
    'latency': 100,    # 延迟阈值（毫秒）
    'packet_loss': 5,  # 丢包率阈值（百分比）
    'port_status': True  # 端口状态变化告警
}
```

### 监控间隔配置
可以调整监控检查的时间间隔：
```python
self.check_interval = 5  # 检查间隔（秒）
```

## 使用方法

1. 启动程序：
```bash
python ip_monitor.py
```

2. 访问Web界面：
```
http://localhost:5000
```

3. 添加监控IP：
   - 在输入框中输入IP地址
   - 可选输入要监控的端口（用逗号分隔，如：80,443,8080）
   - 点击"添加"按钮

4. 查看监控数据：
   - 延迟趋势标签页：查看IP延迟变化趋势
   - 端口状态标签页：查看端口状态变化
   - 告警历史标签页：查看历史告警记录

5. 导出数据：
   - 点击"导出数据"按钮下载CSV文件
   - CSV文件包含完整的监控数据

## 数据说明

### 监控数据
- 延迟：IP的响应时间（毫秒）
- 丢包率：数据包丢失的百分比
- 端口状态：开放/关闭
- MTR结果：网络路由信息

### 告警数据
- 告警时间
- 告警类型
- 告警详情
- 告警状态

### 导出数据格式
CSV文件包含以下字段：
- 时间戳
- IP地址
- 延迟
- 丢包率
- 端口状态
- 告警信息

## 常见问题

1. MTR工具未安装
   - 确保已正确安装MTR工具
   - 检查系统环境变量是否正确配置

2. 端口监控失败
   - 检查防火墙设置
   - 确认目标端口是否开放
   - 验证网络连接是否正常

3. 告警未触发
   - 检查告警阈值配置
   - 确认监控数据是否正常采集
   - 查看系统日志获取详细信息
